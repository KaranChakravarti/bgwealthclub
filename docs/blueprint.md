# **App Name**: BG Wealth Hub

## Core Features:

- Wealth Education: Provide structured learning paths on wealth creation and financial discipline.
- Investment Opportunities Board: Curate and display wealth-sharing and investment opportunities with clear risk disclosures.
- Community Forum: Enable members to connect, share insights, and support each other.
- Financial Guidance Summaries: AI powered summaries of financial guidelines, updates, and support materials using the information from a document provided as a tool for reasoning.
- Membership Management: Handle user registration, profiles, and access control.
- Testimonial Display: Showcase authentic testimonials to build trust.
- Contact and Support: Provide multiple channels for users to reach out for support and guidance.

## Style Guidelines:

- Primary color: Dark blue (#1E3A8A) to inspire trust and professionalism.
- Background color: Very light blue (#F0F9FF) to provide a clean and open feel.
- Accent color: Gold (#D4AF37) to suggest wealth and prosperity.
- Body and headline font: 'Inter' sans-serif font.
- Use modern, professional icons that convey financial concepts clearly.
- Mobile-first responsive design with a clean, minimal aesthetic.
- Subtle transitions and animations to enhance user experience without being distracting.
